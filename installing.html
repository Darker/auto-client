Download

To start download, use the download button to the top. If you want to try beta versions with cutting-edge features, click here..

To find your downloaded file, press {%include kbd.html keys="Ctrl + J"%} and then click on the folder icon in the popup:

Unpack

To unpack files, you can use windows. Double click the archive and you will see the files inside: Open second window in Windows Explorer ( +E) and drag all files to any folder you want:

Install java 8 (if needed)

Check if you have Java 8 or higher. If not, download java here: java.com/en/download/.

Run program

Once Java is installed, you can run the program by double clicking AutoClient.jar.
